Overview

The EmergobyUL.com Visual System provides a flexible foundation for innovation and brand expression. The system is used to communicate the Emergo brand and its products in a unified and consistent manner. Below is a guide to the various design elements that make up our visual identity.

Primary brand colors

Business unit color application

Specific colors are applied business units to visually distinguish one from another.

Extended colors

The extended color pallete is borrowed from UL master brand palette. These extended colors are available but should be used sparingly.

Font Family

Font System

These specific values are used to demonstrate the type system. Values on the website may vary because of responsive behaviors.

Iconography

Leveraging UL's enterprise iconography styling, icons should remain simple and easy to understand, supporting content with minimal distraction or adornment.

Grid System - Desktop

The responsive website uses a flexible set of columns to define content bounds. This grid system allows for maximum flexibilty across devices and resolutions.

Grid System - Tablet

Grid System - Mobile

Navigation

Navigation should remain simple and easy to understand at a glance. Complex labels should be avoided.

Navigation - Drawer

The navigation drawer allows access to important content under each navigation point. The drawer clearly provides a title and its accompanying description, sub categories, and featured area.

Header Level 2

The heder area provides customer with the main context for the page they are viewing. The title, description, and main call to action set the tone and prupose of the rest of the page.

Header Dont's

The header should maintain brand colors and legibiilty. Avoid light and overly complex backgrounds that lead to legibility issues. Backgrounds with text should be avoided.
